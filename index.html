Available Commands

/addtask [task description] - Create and assign a new task
/showlist - Display all tasks with status controls

Setup Instructions

  1. Go to https://api.slack.com/apps and create a new Slack app
  2. Add the following slash commands:
    Command: /addtask
    Request URL: [YOUR_DOMAIN]/slack/addtask
    Command: /showlist
    Request URL: [YOUR_DOMAIN]/slack/showlist
  3. Add the following interactive component:
    Request URL: [YOUR_DOMAIN]/slack/interactive
  4. Install the app to your workspace
  5. Set your Slack signing secret as an environment variable

How It Works

  1. Create Task: Use /addtask [description] to create a new task
  2. Assign Member: Click on a team member button to assign the task
  3. View Tasks: Use /showlist to see all tasks with dates and status
  4. Update Status: Click status buttons to change task progress (dispatcher control)
