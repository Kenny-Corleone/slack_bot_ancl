Privacy Policy - Task Assignment Bot

Information We Collect

Task Assignment Bot collects only the following information:

  • Task descriptions and assignments created by users
  • Channel IDs where tasks are created
  • User IDs of task creators and assignees
  • Task status updates

How We Use Information

We use this information solely to:

  • Display tasks in Slack channels and Home Tab
  • Track task status and assignments
  • Provide task management functionality

Data Storage

All data is stored securely in our database and is not shared with third parties.

Contact

For privacy questions, contact: support@taskassignmentbot.com
