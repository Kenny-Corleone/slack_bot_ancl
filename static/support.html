Support - Task Assignment Bot

📋 Available Commands

/addtask [описание]
Create a new task and assign it to a team member
/showlist
Display all tasks in a table format with Date, Task, Referred, and Status columns
/createtaskchannel
Get instructions for creating a dedicated task channel

🚀 Features

📊 Table Format

Tasks are displayed in a clean table format:

Date           | Task                    | Referred | Status
---------------|-------------------------|----------|--------
09-Jul-2025    | Review budget report   | David    | ✅ Done
09-Jul-2025    | Update documentation   | Emma     | 🔄 In Progress

🏠 Home Tab Integration

View your tasks directly in the app's Home Tab without cluttering chat channels.

👥 Team Management

Assign tasks to team members: David, Emma, Nora, Eric, Kenny

📈 Status Tracking

Track task status: ✅ Done, 🔄 In Progress, ❌ Not Done
